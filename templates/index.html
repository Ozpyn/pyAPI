API Documentation

Endpoints:

  • Create:
    • POST /api/newVehicle
    • POST /api/newCustomer
    • POST /api/newOrder
  • Read:
    • GET /api/getAllVehicles
    • GET /api/getVehicle/{vin}
    • GET /api/getVehicleFeatures/{vin}
    • GET /api/getVehiclePhotos/{vin}
    • GET /api/getCustomer/{id}
    • GET /api/getAllCustomers
    • GET /api/getOrder/{id}
    • GET /api/getAllOrders
  • Update:
    • UpdateCustomerPhoneNumber
    • UpdateCustomerAddress
    • UpdateVehicle
    • UpdateOrder
  • Delete:
    • DELETE /api/deleteVehicle/{vin}
    • Delete Customer
    • Delete Order
  • Base endpoints:
    • GET / - Welcome Page

Response Format:

The API returns JSON data in the following format:

{
    "success": "Vehicle added successfully"
}

or

{
    "error": "Internal Server Error"
}

depending on the outcome of the request.

Sample API Calls

Create Endpoints:

  • Create a New Vehicle:

                POST /api/newVehicle
            Content-Type: application/json
            
            {
                "vin": "ABC123",
                "year": 2022,
                "color": "Red",
                "mileage": 5000,
                "make": "Toyota",
                "model": "Camry",
                "type": "Sedan",
                "mpg-city": 28,
                "mpg-hwy": 35,
                "msrp": 25000,
                "photos": ["photo1.jpg", "photo2.jpg"],
                "features": ["Feature 1", "Feature 2"]
            }

  • Create a New Customer:

                POST /api/newCustomer
            Content-Type: application/json
            
            {
                "email": "customer@example.com",
                "first_name": "John",
                "last_name": "Doe",
                "street_name": "Main St",
                "street_number": "123",
                "city": "Anytown",
                "state": "CA",
                "zip_code": "12345",
                "phone_numbers": ["123-456-7890", "987-654-3210"],
                "phone_types": ["Home", "Work"]
            }

  • Create a New Order:

                POST /api/newOrder
            Content-Type: application/json
            
            {
                "vin": "ABC123",
                "customer_id": 1,
                "street_name": "Main St",
                "street_number": "123",
                "apartment_number": "",
                "city": "Anytown",
                "state": "CA",
                "zip": "12345"
            }

  • Create a New Ownersip Relation:

                POST /api/newOwner
            Content-Type: application/json
            
            {
                "vin": "ABC123",
                "customer_id": 1,
            }

Read Endpoints:

  • Get All Vehicles:

     
                GET /api/getAllVehicles

  • Get Vehicle by VIN:

     
                GET /api/getVehicle/ABC123

  • Get Vehicle Features by VIN:

     
                GET /api/getVehicleFeatures/ABC123

  • Get Vehicle Photos by VIN:

     
                GET /api/getVehiclePhotos/ABC123

  • Get Customer by ID:

     
                GET /api/getCustomer/1

  • Get All Customers:

     
                GET /api/getAllCustomers

  • Get Order by ID:

     
                GET /api/getOrder/1

  • Get All Orders:

     
                GET /api/getAllOrders
