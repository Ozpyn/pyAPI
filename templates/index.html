API Documentation

This is a simple API documentation for the Vehicle API.

Endpoints

1. Get all vehicles

GET /api/getAllVehicles

Retrieves information about all vehicles.

2. Get vehicle by VIN

GET /api/getVehicle/<vin>

Retrieves information about a vehicle based on its VIN.

3. Get vehicle features by VIN

GET /api/getVehicleFeatures/<vin>

Retrieves features of a vehicle based on its VIN.

4. Get vehicle photos by VIN

GET /api/getVehiclePhotos/<vin>

Retrieves photos of a vehicle based on its VIN.

5. Delete vehicle by VIN

GET /api/deleteVehicle/<vin>

Deletes a vehicle from the database based on its VIN.

6. Create new vehicle

POST /api/newVehicle

Creates a new vehicle entry in the database. Requires JSON payload with the following fields: vin, year, color, mileage, make, model, type, mpg-city, mpg-hwy, msrp, photos (optional), features (optional).

7. Create new order

POST /api/newOrder

Creates a new order in the database. Requires JSON payload with the following fields: vin, customer_id, street_name, street_number, apartment_number, city, state, zip.

Example Usage

Assuming the API is hosted on http://174.104.199.92:5004:

        
            // Get all vehicles
            GET http://174.104.199.92:5004/api/getAllVehicles

            // Get vehicle by VIN
            GET http://174.104.199.92:5004/api/getVehicle/ABC123

            // Get vehicle features by VIN
            GET http://174.104.199.92:5004/api/getVehicleFeatures/ABC123

            // Get vehicle photos by VIN
            GET http://174.104.199.92:5004/api/getVehiclePhotos/ABC123

            // Delete vehicle by VIN
            GET http://174.104.199.92:5004/api/deleteVehicle/ABC123

            // Create new vehicle
            POST http://174.104.199.92:5004/api/newVehicle
            {
                "vin": "ABC123",
                "year": 2022,
                "color": "Blue",
                "mileage": 15000,
                "make": "Toyota",
                "model": "Camry",
                "type": "Sedan",
                "mpg-city": 25,
                "mpg-hwy": 35,
                "msrp": 25000,
                "photos": ["photo1.jpg", "photo2.jpg"],
                "features": ["Feature 1", "Feature 2"]
            }

            // Create new order
            POST http://174.104.199.92:5004/api/newOrder
            {
                "vin": "ABC123",
                "customer_id": 1,
                "street_name": "Cedar Bark Drive",
                "street_number": 675,
                "apartment_number": 1,
                "city": "Aurora",
                "state": "Ohio",
                "zip": 44202
            }
